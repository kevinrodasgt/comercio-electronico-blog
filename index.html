EN QUE NOS AYUDO EL CSS

La css nos ayudo bastante ya que crea reglas a nuestro sitio web, como quiere mostrar la información y ademas guarda los comandos para los elementos de estilo como los colores, fuentes, tamaño etc.

con los elementos de la pagina web que creamos con html se les dara la apariencia que se desea utilizanto CSS, Esta separacion entre la estructura y la presentacion es muy importante, ya que permite que sólo los CSS se modifique completamente el aspecto de nuestra pagina,

EN QUE NOS AYUDO HTML

html es el lenguaje que emplea para el desarrollo de paginas de internet, está compuesto por una serie de etiquetas que el navegador interpreta y da forma en la pantalla .

en html nos ayudo a la estructura de nuestra pagina por ejemplo podemos crear un parrafo colocando el texto encerrado dentro de una etiqueta inicial.

Google analitics
